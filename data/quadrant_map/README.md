# Quadrant Map — data

Unlike the domestic-wells notebook, this analysis needs **no manual data download**. All inputs
are fetched live when the notebook runs:

- **Area of interest** — Census Bureau "Albuquerque, NM Urban Area" boundary (GEOID `01171`),
  fetched from the public TIGERweb REST API (no account needed).
- **Water-use rasters** — OpenET ensemble evapotranspiration and GRIDMET precipitation, both
  pulled from Google Earth Engine. This does require your own free Earth Engine account and
  Google Cloud project — see the notebook's setup section
  (`../../notebooks/quadrant_map/quadrant_map_replication.ipynb`) for the five-minute signup.

## `output/`

The notebook writes its downloaded GeoTIFFs and rendered PNG here. Not committed to the
repository (regenerated by running the notebook).
