# Rendered Notebooks

These are HTML renderings of the Jupyter notebooks, converted automatically from `.ipynb` files on push.

The notebooks are fully executable—to run them:
1. Clone this repository
2. Install dependencies (`pip install -r ../requirements.txt` for Python, or R packages for R notebooks)
3. Open the `.ipynb` file in Jupyter, JupyterLab, or your notebook environment
4. Run cells in order

The HTML versions here are read-only but include all outputs, charts, and tables from the last successful run.

**Note:** Download links in the HTML versions may not work. For full execution, run the notebook locally.
