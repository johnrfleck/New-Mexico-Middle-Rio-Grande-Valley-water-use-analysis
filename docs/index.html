Water Use Analysis: New Mexico’s Middle Rio Grande Valley#

New Mexico’s Middle Rio Grande Valley faces acute water management challenges. Rio Grande inflow is declining, consumptive use of water is increasing, and large uncertainties - both hydrologic and policy-related - face efforts to bring supply and use into balance.

This project documents an effort to bring new tools to bear on the problem, to better inform decision making. Based in an interdisciplinary effort among students and faculty from the University of New Mexico Water Resources Program, the UNM Department of Economics, and the Utton Transboundary Resources Center at the University of New Mexico School of law, this open research project makes the accounting and data behind water-use analyses transparent and reproducible. All notebooks are executable—download the repo, install dependencies, and run the analyses yourself.

What We’re Studying#

The Middle Rio Grande Valley faces competing demands for water from agriculture, cities, and the river ecosystem. Understanding how water is actually used—and how that’s changed over time—is essential for informed policy.

This repository houses analyses of:

  • Domestic wells: How many active domestic wells are registered in the region, and how much water do they have permission to use?

  • Net Flow Reduction: What happens to river flow as it moves downstream? Natural processes, human diversions, and measurement challenges all play a role.

How to Use This Site#

Each analysis includes:

  1. Methodology — Plain-language explanation of questions, approach, and caveats

  2. Interactive notebook — Executable Python or R code that reproduces the analysis

  3. Published figures — Key results and charts

Click the chapters in the sidebar to explore each analysis. Code cells are hidden by default—click “toggle” to see the implementation details.

Running Notebooks Locally#

All notebooks are fully executable. To replicate an analysis:

  1. Clone the repository: git clone https://github.com/johnrfleck/New-Mexico-Middle-Rio-Grande-Valley-water-use-analysis.git

  2. Install dependencies: pip install -r requirements.txt (Python) or the R packages listed in each notebook

  3. Open a notebook in Jupyter, JupyterLab, or your notebook environment

  4. Run cells in order

Large spatial datasets (shapefiles) must be downloaded separately — see each notebook’s data section for links.

Status#

This is an active public working repository. Material is shared early to make analyses easier to inspect, question, and improve. Interpretive claims are provisional unless directly supported by the notebooks and documentation.

Questions? File an issue on GitHub.
