# Domestic Wells — output data

Derived products regenerated by the domestic-wells notebook on every run:

- `domestic_wells_counting_ladder.csv` — the four-rung counts (active PODs → physical wells →
  unique rights → solidly-inside) and permitted AF for MRGCD and ABCWUA.
- `domestic_wells_cumulative_time_series.csv` — new and cumulative unique rights and permitted AF by
  completion year, 1950–2025, per boundary.
- `cumulative_permitted_diversion.png` — the cumulative-diversion chart.

These are git-ignored regenerated products; they are not committed. The executed notebook in
`notebooks/domestic_wells/` carries the same tables and chart inline as its cell outputs.
