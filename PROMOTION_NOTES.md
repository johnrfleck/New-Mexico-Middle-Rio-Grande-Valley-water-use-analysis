# Promotion Notes

This file records what should move from the private working tree into the public package.

## Promotion Rule

Move only approved artifacts into this directory.

Do not treat this directory as a mirror of `NCU/`.

## Expected Inputs

- selected notebooks from `public_notebooks/` and `public_notebooks_r/`
- selected CSV files from `Hydro Data/`
- selected explanatory documents rewritten for public readers
- selected Google Earth Engine scripts or notes after source and export assumptions are documented

## Before Copying

- confirm the notebook or file supports the public writeup directly
- confirm there are no local paths, private notes, or hidden environment assumptions
- confirm any figures or outputs are intentionally retained
- confirm the item has been reviewed in `review_bench/`
